# Product Builder JHR

## Requirements

- [Miniconda](https://docs.conda.io/en/latest/miniconda.html) or any Python package manager.
- Python 3.13

---

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/zen920/product-builder-jhr.git
cd product-builder-jhr
```

### 2. Create and activate the conda environment

```bash
conda create -n ProductBuilderJHR python=3.13
conda activate ProductBuilderJHR
```

### 3. Install the package

> **Important:** Use `python -m pip` instead of `pip` directly to ensure the correct Python environment is used.
```bash
python -m pip install .
```

### 4. Run the first-launch setup

```bash
python -m product_builder_jhr.scripts.setup
```

This will generate the required resources and configuration files.

## Running the App

> **Important:** Run from the main directory of the project (the one created via clone).

```bash
streamlit run "src/product_builder_jhr/pages/main.py"
```

---

## Development

For development, install in editable mode so code changes are reflected immediately without reinstalling:

```bash
python -m pip install -e .
```

### Running tests

```bash
python -m pytest tests/ -v
```
